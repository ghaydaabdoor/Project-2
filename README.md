# Project-2
Refurn - Used Furniture Online Marketplace!

Welcome to Refurn, your trusted destination for high-quality, preloved furniture in Jordan. This repository contains the wireframes and mockups for my website, designed to provide a seamless and enjoyable online shopping experience.

Project Overview
Refurn is an online marketplace where customers can browse and buy a range of used furniture. My website is designed to be user-friendly and aesthetically pleasing, with a focus on sustainability and affordability.

Features:
Home Page: Highlights the unique value we offer.
Register Page: Allows new users to create an account easily.
Login Page: Enables returning users to access their accounts.
Categories: contains categorized furniture to buy.
Service Page: Displays our product listings, including detailed descriptions and images.
Checkout Page: Streamlines the purchase process for our customers
